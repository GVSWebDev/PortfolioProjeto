Lampada Gamer

2020-05-31 Joao Larini
Em 2017 comecei um curso para aprender mais sobre o hardware do computador e servidores e seus usos/configuração. No final do curso precisei fazer um trabalho para conclusão utilizando todos os assuntos ensinados. Minha decisão foi fazer uma lâmpada/abajur ser ligado remotamente pelo celular ou computador. Para isso utilizei um micro computador chamado Raspberry PI [Imagem 1], ele possui vários barramentos idênticos ao de um computador normal (USB, Ethernet, etc) e utiliza um micro SD como armazenamento.
